1. ASYMPTOTY
────────────────────────────────────────────────────────────────────────────────
POZIOMA:    y = lim(x→±∞) f(x)

UKOŚNA:     y = ax + b
            a = lim(x→±∞) f(x)/x
            b = lim(x→±∞) [f(x) - ax]

PIONOWA:    x = x₀, gdzie lim(x→x₀) f(x) = ±∞

Przydatne:  arctg(x) → π/2  gdy x→+∞
            arctg(1/t) = π/2 - arctg(t)  dla t>0

────────────────────────────────────────────────────────────────────────────────
2. CAŁKI - UŁAMKI PROSTE
────────────────────────────────────────────────────────────────────────────────
1) Rozłóż mianownik na czynniki
2) Rozkład:  P(x)/[(x-a)(x²+bx+c)] = A/(x-a) + (Bx+C)/(x²+bx+c)
3) Wyznacz A,B,C przez podstawianie lub porównywanie współczynników

Wzory:
∫ 1/(x+a) dx = ln|x+a| + C
∫ 1/(x²+a²) dx = (1/a)·arctg(x/a) + C
∫ x/(x²+a²) dx = (1/2)·ln(x²+a²) + C

────────────────────────────────────────────────────────────────────────────────
3. DŁUGOŚĆ ŁUKU
────────────────────────────────────────────────────────────────────────────────
L = ∫[a,b] √(1 + (y')²) dx

Parametrycznie:  L = ∫[t₁,t₂] √((x')² + (y')²) dt

────────────────────────────────────────────────────────────────────────────────
4. OBJĘTOŚĆ BRYŁY OBROTOWEJ
────────────────────────────────────────────────────────────────────────────────
Obrót wokół OX (metoda krążków):
    V = π ∫[a,b] y² dx

Obrót wokół OY (metoda powłok):
    V = 2π ∫[a,b] x·y dx

────────────────────────────────────────────────────────────────────────────────
5. SZEREGI POTĘGOWE - PROMIEŃ ZBIEŻNOŚCI
────────────────────────────────────────────────────────────────────────────────
Σ aₙ(x-x₀)ⁿ

d'Alembert:   r = lim(n→∞) |aₙ/aₙ₊₁|
Cauchy:       r = 1 / lim(n→∞) ⁿ√|aₙ|

Przydatne:    lim(n→∞) (1 + 1/n)ⁿ = e
              lim(n→∞) ⁿ√n = 1
              n! ≈ √(2πn)·(n/e)ⁿ  (Stirling)

────────────────────────────────────────────────────────────────────────────────
6. EKSTREMA FUNKCJI DWÓCH ZMIENNYCH f(x,y)
────────────────────────────────────────────────────────────────────────────────
1) Oblicz:  fₓ = ∂f/∂x,  fᵧ = ∂f/∂y
2) Rozwiąż układ:  fₓ = 0,  fᵧ = 0  → punkty krytyczne
3) Oblicz:  fₓₓ, fᵧᵧ, fₓᵧ
4) Hesjan:  H = fₓₓ·fᵧᵧ - (fₓᵧ)²

INTERPRETACJA:
• H > 0  i  fₓₓ > 0  →  MINIMUM
• H > 0  i  fₓₓ < 0  →  MAKSIMUM
• H < 0              →  PUNKT SIODŁOWY
• H = 0              →  NIE ROZSTRZYGA

────────────────────────────────────────────────────────────────────────────────
7. CAŁKI NIEWŁAŚCIWE
────────────────────────────────────────────────────────────────────────────────
I rodzaju (nieskończone granice):
∫[a,∞) f(x)dx = lim(t→∞) ∫[a,t] f(x)dx

II rodzaju (osobliwość w b):
∫[a,b] f(x)dx = lim(ε→0⁺) ∫[a,b-ε] f(x)dx

Kryterium porównawcze:
∫[1,∞) 1/xᵖ dx  zbieżna ⟺ p > 1
∫[0,1] 1/xᵖ dx  zbieżna ⟺ p < 1

8. POCHODNE CZĄSTKOWE - WZORY
────────────────────────────────────────────────────────────────────────────────
∂/∂x(ln y) = 0           ∂/∂y(ln y) = 1/y
∂/∂x(xⁿ) = n·xⁿ⁻¹        ∂/∂y(yⁿ) = n·yⁿ⁻¹
∂/∂x(xy) = y             ∂/∂y(xy) = x

────────────────────────────────────────────────────────────────────────────────
9. GRANICE - REGUŁA DE L'HÔPITALA
────────────────────────────────────────────────────────────────────────────────
Dla 0/0 lub ∞/∞:  lim f(x)/g(x) = lim f'(x)/g'(x)

Typ ∞·0:  zamień na ułamek i użyj de l'Hôpitala
Typ 1^∞:  lim f^g = e^(lim g·ln f)

────────────────────────────────────────────────────────────────────────────────
10. SZEREG TAYLORA / MACLAURINA
────────────────────────────────────────────────────────────────────────────────
f(x) = Σ f⁽ⁿ⁾(a)/n! · (x-a)ⁿ

eˣ = 1 + x + x²/2! + x³/3! + ...
sin x = x - x³/3! + x⁵/5! - ...
cos x = 1 - x²/2! + x⁴/4! - ...
ln(1+x) = x - x²/2 + x³/3 - ...      |x| < 1
1/(1-x) = 1 + x + x² + x³ + ...      |x| < 1
